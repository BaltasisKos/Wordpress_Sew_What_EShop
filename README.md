# 🪡 Sew What Website (WordPress Project)

Welcome to the repository for the **Sew What** website. This project is a fully dynamic WordPress website developed to showcase custom sewing, handmade creations, alterations, and design services.

I created this project to demonstrate my skills in WordPress development, custom staging/configuration, and advanced frontend presentation.

## 🎯 Project Overview

The main goal of this project was to build a modern, responsive, and visually appealing website for a sewing and crafting business. It serves as a digital portfolio and service hub where clients can browse handmade products, view available services, and get in touch for custom orders.

### Key Features:
*   **Advanced Motion & Animations:** Powered by GSAP for smooth, high-end visual transitions and engaging user interactions.
*   **Dynamic Portfolio/Gallery:** A clean showcase of custom sewing projects and handmade items.
*   **Service Catalog:** Easily manageable list of services, pricing, and alteration options.
*   **Fully Responsive:** Optimized for seamless browsing on mobile devices and tablets.
*   **SEO & Performance Optimized:** Built with best practices to ensure fast loading times and discoverability.

## 🛠️ My Role & Tech Stack
